# cartrend — 신차장기렌트 전문 브랜드 홈페이지

신뢰감 있고 프리미엄한 **신차장기렌트** 전문 브랜드 **cartrend**의 원페이지 랜딩 사이트입니다.
초기비용 0원·월 렌트료 중심으로, 토스·카카오모빌리티 수준의 깔끔한 SaaS 스타일 UI, 모바일 우선 반응형, 빠른 견적 전환에 초점을 맞췄습니다.

## 실행 방법
빌드 도구 없이 정적 파일입니다. 그냥 열면 됩니다.

```bash
open index.html          # macOS
# 또는 로컬 서버
python3 -m http.server 8080   # → http://localhost:8080
```

## 구성 파일
- `index.html` — 마크업 (10개 섹션 + 헤더/푸터/플로팅 예약 버튼)
- `styles.css` — 컬러 시스템·레이아웃·반응형·애니메이션
- `script.js` — 차량 SVG 일러스트, 데이터 렌더링, 탭/슬라이더/아코디언/스크롤 애니메이션

## 컬러 시스템
| 역할 | 값 |
|---|---|
| 메인 (프리미엄 오렌지) | `#F57C00` |
| 서브 (다크 네이비) | `#1A2332` |
| 포인트 (라이트 오렌지) | `#FFB74D` |
| 배경 | `#FFFFFF` / `#F7F8FA` |

## 섹션
1. 히어로 — 카피 + CTA(견적받기/전화상담) + 차량 비주얼
2. 빠른 견적 — 차종/계약기간/선납금(보증금)/약정 주행거리 + 견적받기 (히어로 위로 오버랩)
3. 인기 차량 BEST — 카테고리 탭(경차·준중형·중형·대형·SUV·승합차·수입차) + 월 렌트료 카드
4. 신차장기렌트 진행 절차 4단계 (차량선택 → 견적·상담 → 계약접수 → 신차출고)
5. 왜 우리를 선택해야 할까요 — 초기비용 0원·합리적 월 렌트료·정비/보험 올인원·만기 옵션 자유선택
6. 상황별 추천 차량 — 가로 슬라이더
7. 고객 후기 — 카드 UI
8. 자주 묻는 질문 — 아코디언 (장기렌트 기준)
9. 상담 CTA 배너 (오렌지)
10. 푸터

## 참고
- 차량 이미지는 외부 의존성 없이 인라인 SVG 일러스트로 제작했습니다. 실제 차량 사진으로 교체하려면 `script.js`의 `carSVG()` 호출부를 `<img>`로 바꾸면 됩니다.
- 폰트는 Pretendard(CDN)를 사용합니다.
- 견적/상담 버튼은 데모용 `alert`으로 동작하며, 실제 견적·상담 API 연동 지점만 바꾸면 됩니다.
- 월 렌트료는 예시 금액(선납·계약기간·주행거리에 따라 변동)입니다. 실제 운영 시 `script.js`의 `CARS` 데이터를 교체하세요.

> 아마존카의 **신차장기렌트 차량 구성 방식과 카테고리 분류만 참고**했으며, 레이아웃·디자인·UI/UX는 전부 새로 제작했습니다.
